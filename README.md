# MacroPad
This is a small **4-key RGB macro pad** project built using the Seeed Studio XIAO RP2040.  
It is designed as a beginner-friendly hardware + firmware project and can be used as a USB macro pad for shortcuts, media control, or productivity.


---

##  Features

- 4 mechanical keys
- Per-key RGB lighting using SK6812 MINI LEDs
- Rotary encoder for volume / Brightness
- OLED display for status 
- USB HID 
- Compact custom PCB
- Customizable Code For Customized Features

## How To Use Rorary Encoder 

- Double Click to switch btw brightness and volume
- Scroll to adjust
- Tripple Click To Switch RGB Modes
- Single Click For Play Pause

   CODE CAN BE EDITED AS PER CHOICE
